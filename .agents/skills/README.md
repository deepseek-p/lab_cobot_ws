# 项目级 ROS 2 技能

本目录为 `lab_cobot_ws` 的项目级 Agent Skills，仅服务于当前 ROS 2 Humble、Gazebo Classic、Nav2、MoveIt 2、RGB-D 感知和麦轮底盘工程。

## 已选技能（20 个）

- 工作区与基础设施：`colcon-workspace`、`launch-files`、`coordinate-frames-and-tf`、`logging-and-diagnostics`
- 测试：`ros2-testing`、`robotics-testing`
- URDF/Gazebo：`collision-geometry`、`inertia-calculation`、`physics-tuning`、`urdf-gz-plugins`
- 控制与导航：`controller-frequency-tuning`、`nav2-simple-commander`、`dwb-controller`、`costmap-architecture`、`ekf-sensor-fusion`
- 感知：`depth-camera-pipeline`、`object-detection-pipeline`、`robot-perception`
- 系统工程：`robot-bringup`、`ros2-engineering-skills`

## 版本注意事项

`ros2-copilot-skills` 主要面向 ROS 2 Jazzy/Rolling。本项目使用 ROS 2 Humble；涉及参数名、插件名和 API 时，必须以本项目源码及 Humble 已安装接口为准。
